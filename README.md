# SamAlS
Хлебопечка на Ардуино
Проект хлебопечкпи на Ардуино изначально параметры программ выпечки считаны в логи и проанализированы температура и алгоритм срабатывания ТЭНа и мотора. Для считвания ипользовались контакты реле с катушкой на 220 вольт подпаянных к ТЭНу и мотору, термистор 100 кОм в составе сенсора от термометра.
Изначально считывались параметры с печки LG-151.
Считыватель частично позаимствован у другого автора, ссылка на него в шапке файла в папке bm_recorder_100.
Схема в файлах 1 и 2.
